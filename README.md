# BashISE
BashISE (bise2) is a local-first, package-first GUI editor + Bash runner for Linux. Includes project tree, Quick Open, find/search, Run/Run Selection, Check and evidence-pack export (receipts/logs/hashes). Ships as wheel with a per-user runtime venv installer (no sudo) and safety guardrails via root scoping + deny-roots. Built by Trishula Software.
